CSS Selectors

Introduction

CSS adds the style to the html page. It consists of a series of rules that specify how each element should be displayed. A CSS rule contains two parts: a selector and a declaration.

CSS Selectors

CSS selectors can be used to target all elements of the same type, elements with a specific class, or elements with an id.

CSS selectors can be nested and can have all sorts of combinations to target a specific element
